# PayRouter static website mock

This is a dependency-free static build of the PayRouter website and product-dashboard mock from the brief.

## Run locally

```bash
npm start
```

Then open:

```text
http://localhost:4173
```

## Included routes

- `/`
- `/dashboard`
- `/compare`
- `/recommendation`
- `/transactions`
- `/settings`

All data is mock/demo data and no backend is required.
